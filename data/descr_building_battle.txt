	; Format: 
	; orig_texturename
	; {
	;    <season or faction or culture, or any combination> variant_name
	; }
	; ROME HD:
	;		When no faction or culture is specified we now apply to all factions.
	;		When no filters or conditions of any sort are wanted, `any` is now accepted as a wildcard.
	;		In addition to the faction or season specifiers, a single condition type can be added like so:
	;		orig_texturename
	;		{
	;			<1 season or faction or culture or condition, or any combination XOR any> variant_name
	;		}
	;		When no condition is applied then the behaviour is the same as before.
	;		The following conditions are allowed:
	;			"snowfall" or "blizzard" - Make this texture only appear when it is snowing.
	;			"snowcover" - Make this texture only appear when snow covers the ground (generally winter + climate has [snow_in_]winter flag).
	;		Where multiple variants apply, the final one will win, so place conditional variants after generic variants
	; Note factions must be the underscored versions, ie no spaces.  Also, no spaces allowed in texture names.
	; eg:
	; #roof.tga
	; {
	;    winter #roof_winter.tga
	; }
